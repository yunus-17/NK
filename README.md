# NK Engineering Hub

Premium Structural Engineering and Asset Management Platform.

## Architecture
- **Frontend**: React, Tailwind CSS, Framer Motion
- **Backend**: Node.js, Express, MongoDB, JWT Authentication
- **Features**: Real-time project tracking, Administrative Command Center, Secure Client Enquiries.

## Setup
1. Clone the repository.
2. Install dependencies in both `frontend` and `backend` directories.
3. Configure `.env` in the `backend`.
4. Run `npm dev` for frontend and `npm start` for backend.
